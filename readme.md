Project Setup Guide

Before running the project, make sure to create a .env file in the root directory of your project and add the following environment variables:

DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
DB_DIALECT=mysql
PORT=
JWT_SECRET=

After setting up the environment variables, run the following commands to install dependencies and start the server:

npm install
npm start

For Frontend 

add .env file in app folder and add the following environment variables:

VITE_API_URL= your api urlW

then run the following commands to install dependencies and start the frontend:

npm install
npm run dev
